# Mining Pistol

**Mining Pistol** is the standard starting weapon in COSMIC KNIGHTS, designed
for [[cosmic-knights-crystal-economy|crystal extraction]] and basic combat while
serving as the foundation for Banner tactical coordination.

## Weapon Overview

The Mining Pistol represents **dual-purpose utility technology**, serving both
as a resource extraction tool and a reliable combat weapon for beginning Knights
and tactical operations.

## Core Mechanics

### Crystal Extraction

- **Crystal Mining**: Primary function for extracting
  [[cosmic-knights-crystal-economy|Crystal]] resources from environmental
  sources
- **Precision Extraction**: Controlled extraction minimizing crystal waste and
  environmental damage
- **Efficiency Systems**: Optimized extraction providing maximum crystal yield
  from available sources
- **Environmental Integration**: Extraction capability working with various
  crystal formation types

### Combat Capability

- **Ballistic Function**: Reliable projectile weapon for basic combat and
  tactical engagement
- **Rapid Fire**: Quick firing capability for responsive combat and defensive
  applications
- **Accuracy Systems**: Precision targeting for both combat and extraction
  applications
- **Reliability Design**: Durable construction ensuring consistent performance
  under various conditions
