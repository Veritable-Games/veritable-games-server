# ON COMMAND Character Classes and Specializations

## Class Design Philosophy

### Accessibility with Specialization

- **Universal Capability**: Everyone can perform any task for game accessibility
- **Individual Excellence**: Each soldier excels at specific tasks
- **Performance Variance**: Each soldier has tasks they're terrible at
- **Balanced Team Composition**: Encourages strategic squad member selection

### Visual Identification System

Each class features distinctive visual elements for immediate battlefield
identification and role clarity.

## Character Class Breakdown

### Technician Specialist

**Primary Focus**: Electronic systems, hacking, and device manipulation

#### Visual Identification

- **Distinctive Element**: Unique optical tool mounted on helmet
- **Purpose**: Immediate visual recognition during combat scenarios

#### Specialized Abilities (Technician Bonus)

- **Setting Traps**: Faster placement and more effective trap deployment
- **Defusing Traps**: Accelerated disarmament of enemy explosive devices
- **Slicing Doors**: Enhanced electronic door breaching capabilities
- **Locking Doors**: Superior door security and access denial systems

#### Tactical Applications

- **Pre-Combat Setup**: Essential for trap-based ambush strategies
- **Breach Operations**: Primary role in electronic door operations
- **Counter-Electronics**: Key to disabling enemy electronic systems
- **Area Denial**: Creates tactical advantages through trap placement

### Strategist Specialist

**Primary Focus**: Positioning, cover manipulation, and battlefield control

#### Visual Identification

- **Distinctive Element**: Third robotic arm mounted on shoulder
- **Functionality**: Enhanced manipulation capabilities for cover construction

#### Specialized Abilities (Strategist Bonus)

- **Cover Setup**: Rapidly establish defensive positions
- **Fortify Cover**: Strengthen existing cover positions for enhanced protection
- **Movement Speed**: Faster transitions between cover positions
- **Position Analysis**: Superior tactical positioning assessment

#### Tactical Applications

- **Defensive Preparation**: Critical for establishing strong defensive lines
- **Mobile Operations**: Enables rapid repositioning during combat
- **Cover Enhancement**: Transforms basic positions into fortified strongpoints
- **Tactical Flexibility**: Adapts battlefield layout to squad advantages

### Enforcer Specialist

**Primary Focus**: Heavy equipment, emplacements, and infrastructure support

#### Visual Identification

- **Distinctive Element**: Specialized nanite dispenser mounted on back
- **Capability**: Enhanced resource distribution and equipment maintenance

#### Specialized Abilities (Enforcer Bonus)

- **Fortify EZ**: Accelerated emplacement zone strengthening
- **Repair EZ**: Enhanced emplacement zone maintenance and restoration
- **Equip EZ**: Faster emplacement zone setup and configuration

#### Tactical Applications

- **Emplacement Management**: Primary responsibility for EZ operations
- **Equipment Maintenance**: Keeps squad equipment operational
- **Heavy Support**: Manages high-value tactical assets
- **Resource Distribution**: Ensures squad has necessary supplies

## Strategic Team Composition

### Optimal Squad Balance

- **One Technician**: Essential for electronic warfare and trap management
- **One Strategist**: Critical for cover establishment and tactical positioning
- **One Enforcer**: Necessary for emplacement management and heavy support
- **Remaining positions**: Fill with complementary personalities and
  capabilities

### Mission-Specific Adaptations

- **Heavy Electronic Warfare**: Multiple Technicians for complex system
  infiltration
- **Defensive Operations**: Multiple Strategists for layered cover systems
- **Emplacement-Heavy Maps**: Multiple Enforcers for equipment-intensive
  scenarios

This character system creates tactical depth while maintaining accessibility and
encouraging strategic thinking about squad composition.
