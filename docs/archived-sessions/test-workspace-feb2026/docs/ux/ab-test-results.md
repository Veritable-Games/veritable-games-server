## Technical Notes

### Analytics Implementation
All variants track the following events:
- `donation_page_viewed` - Page load
- `donation_amount_selected` - User selects preset or custom amount
- `donation_payment_method_selected` - Stripe or BTCPay selected
- `donation_project_selected` - Project allocation chosen
- `donation_form_submitted` - Form submission (before redirect)
- `donation_checkout_started` - Redirect to payment processor
- `donation_error` - Any error during flow

## Appendix

### Test Hypothesis
**Null Hypothesis (H0)**: All variants have equal conversion rates.
**Alternative Hypothesis (H1)**: At least one variant has a significantly different conversion rate.

### Statistical Significance
- **Confidence Level**: 95%
- **Minimum Sample Size**: 100 conversions per variant
- **Test Method**: Chi-square test for conversion rate comparison
